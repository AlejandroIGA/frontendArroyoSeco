# Alojando - Frontend (Arroyo Seco)

**Alojando** es una aplicación web progresiva (PWA) y móvil diseñada para conectar a propietarios de alojamientos con visitantes en la región de Arroyo Seco, facilitando la búsqueda, reservación y gestión de hospedajes.

Este repositorio contiene el código fuente del **Frontend**, desarrollado con React, Ionic y Vite.

## Requisitos Previos

Para ejecutar este proyecto localmente, necesitas tener instalado:

1.  **Node.js**: Versión 20 o superior (recomendado v20 LTS según la configuración de Docker).
2.  **npm**: (Viene instalado con Node.js).
3.  **Git**: Para clonar el repositorio.
4.  **Backend en ejecución**: Necesitas tener el servidor backend de la API corriendo para que la aplicación funcione correctamente.

```bash
git clone <URL_DEL_REPOSITORIO>
cd frontendarroyoseco

Para probar la página web

npm install

npm run dev

Para probar la PWA

npm run preview
